# Box

Proxy for the Box API with direct file uploads on pre-signed folders

## Configuration

- CLIENT_ID
- CLIENT_SECRET
- PUBLIC_KEY_ID
- PRIVATE_KEY
- PASSPHRASE
- ENTERPRISE_ID
- USER_ID
- HMAC_SECRET
